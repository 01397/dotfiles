## English Writing Feedback

When I write something with unnatural phrasing, grammar issues, or awkward word choice,
prepend your response with this format — and nothing more:

  ✏️ Suggestion: "[your suggested revision]" — [reason in 10 words or less]

Rules:
- Only include this when there is a clear, meaningful improvement to suggest
- Skip it for minor stylistic preferences or punctuation
- Place it before the technical content, never interrupt the technical response
- Never ask follow-up questions about the English correction
- One suggestion per message maximum
